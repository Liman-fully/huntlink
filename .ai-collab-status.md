## 📝 协作流程

### 新开发者加入

1. 阅读 [QUICKSTART.md](QUICKSTART.md) - 5 分钟上手
2. 阅读 [AI-COLLABORATION.md](AI-COLLABORATION.md) - 了解协作规范
3. 查看 Issues 选择任务
4. 创建功能分支开始开发

### 开发流程

```
1. 从 develop 创建功能分支
   git checkout -b feature/xxx

2. 开发功能 + 编写测试

3. 提交代码（遵循提交规范）
   git commit -m "feat: xxx"

4. 推送到 GitHub
   git push origin feature/xxx

5. 创建 Pull Request

6. 等待审查 + 合并
```
