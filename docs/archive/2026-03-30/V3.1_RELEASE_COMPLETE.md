## 📊 核心原则（V3.1）

### 🌐 GitHub 为唯一共享空间

```
🌐 GitHub 地址：https://github.com/Liman-fully/huntlink

✅ 所有代码/文档/配置以 GitHub 为唯一可信来源
✅ 每次修改必须立即 push 到 GitHub
✅ 本地内容不可信，GitHub 内容才可信
❌ 禁止依赖任何本地记忆或非官方同步渠道
```

### 📋 执行规则

| 场景 | 操作 | 命令 |
|------|------|------|
| **开始工作前** | 从 GitHub 获取最新代码 | `git pull origin master` |
| **完成任务后** | 立即 push 到 GitHub | `git push origin master` |
| **离开前** | 确保所有代码已 push | `git status` 检查 |
| **新成员加入** | 从 GitHub 克隆完整项目 | `git clone <URL>` |
