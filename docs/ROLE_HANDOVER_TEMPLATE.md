## 🚨 紧急情况处理

### 场景 1: 突然掉线

**症状**：
- 不回复消息
- 职责无人履行
- 组织停滞

**立即行动**：
```
1. 指定临时继任者（由 LEADER 或剩余成员推选）

2. 临时继任者阅读本手册（30 分钟）

3. 发布公告：
   "@全体 [角色] 掉线，我是临时 [角色] [名字]，已接管职责"

4. 按清单执行职责

5. 等待正式继任者
```

### 场景 2: 计划交接

**流程**：
```
1. 提前 [X] 天通知 LEADER
2. 更新本手册（补充最新经验）
3. 培训继任者（[X] 天）
4. 并行工作（[X] 天）
5. 正式交接（1 天）
6. 发布公告
```

### 场景 3: 组织重组

**流程**：
```
1. LEADER 宣布重组计划
2. 重新定义角色
3. 成员选择新角色
4. 完成交接
5. 按新架构运行
```
