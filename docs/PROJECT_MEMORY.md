**时间**: 2026-03-27 22:00-01:43

**问题**：
- 左护法收到 SSH 部署任务
- 没有先搜索技能

**正确做法**：

## 🏆 最佳实践

### 实践 1: 5 分钟闭环流程

**适用场景**：所有开发任务

**流程**：
```
0:00 - 收到任务
0:00-0:30 - 思考（第一性原理）
0:30-1:00 - 技能检查（强制搜索）
1:00-3:00 - 全速开发
3:00-4:00 - 测试
4:00-4:30 - 同步（更新任务看板）
4:30-5:00 - 闭环（提交 + 通知）
```

**效果**：
- ✅ 任务响应时间 ~15 秒
- ✅ 技能使用率 100%
- ✅ 浪费时间 6 倍↓
