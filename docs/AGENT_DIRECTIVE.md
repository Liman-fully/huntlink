# 猎脉 (Huntlink) Agent 归队指令 (V1.1)

## 1. 归队仪式
### 第一步：巅峰战力重载 (Reload Skills)
1. 立即阅读 `docs/SKILL_REGISTRY.md`：查看本项目全栈领域的**最优技能官方清单**。
2. 根据你的任务，从清单中精准 load 以下能力包：
   * 认知类：`prompt-engineering-expert`, `marketing-mode`
   * 执行类：`web-artifacts-builder`, `agent-browser`, `xlsx/pdf`
   * 运维类：`1password`


## 2. 战局同步 (Context Sync)
1. 执行 `cat docs/TODAY_PROGRESS.md` 获取都统最新构建反馈。
2. 执行 `cat DEPLOYMENT_HANDOVER.md` 确认目前修复节点。
3. 执行 `git log -n 3 --oneline` 确认云端同步状态。
