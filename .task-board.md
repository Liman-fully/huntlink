## 📁 设计文档

### 职位发布设计

**位置**: `docs/design/职位发布设计.md`

**核心流程**:
```
粘贴 JD → AI 识别 → 高亮待确认 → 用户修改 → 发布
```

**设计稿**:
```
┌─────────────────────────────────────────┐
│  发布新职位                              │
├─────────────────────────────────────────┤
│  步骤 1: 粘贴 JD 或上传文件              │
│  ┌───────────────────────────────────┐  │
│  │ 高级前端工程师...（粘贴区域）      │  │
│  │ [上传 JD 文件]                     │  │
│  └───────────────────────────────────┘  │
│  [下一步：AI 自动识别]                    │
├─────────────────────────────────────────┤
│  步骤 2: 确认识别结果（高亮待修改）      │
│  职位名称：[高级前端工程师] ✅          │
│  薪资范围：[40-60K] ⚠️（可能不准确）   │
│  工作地点：[北京] ✅                    │
│  经验要求：[5-10 年] ✅                 │
│  ...                                    │
│  [确认发布]                             │
└─────────────────────────────────────────┘
```

**验收标准**:
- ✅ 能粘贴 JD 文本
- ✅ AI 自动识别并填入表单
- ✅ 高亮显示不确定的字段
- ✅ 用户可修改后发布
