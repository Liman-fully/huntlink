## 📝 如何领取任务

### 步骤

1. **查看任务** - 找到"⏳ 待开始"任务
2. **检查最新进展** - 阅读 `docs/TODAY_PROGRESS.md`（避免重复！）
3. **更新执行者** - 改为你的称呼
4. **更新状态** - 改为"🟡 进行中"
5. **添加领取信息**：
   ```markdown
   **领取人**: [你的称呼]
   **领取时间**: 2026-03-28 18:00
   **预计完成**: 2026-03-28 18:30
   **配置**: [记录当前配置]
   ```
6. **提交更新**：
   ```bash
   git add .task-board.md
   git commit -m "task: 领取任务 [任务 ID]"
   git push origin master
   ```
7. **通知协调者**：
   ```
   @协调者 我已领取任务 [任务 ID]，预计 [时间] 完成
   ```

### ⚠️ 同步规则（重要！）

**开发中**：
- 每 5 分钟 → `git pull`（获取最新）
- 每完成一步 → `git commit + git push`（同步进展）
- 遇到报错 → 立即同步 + 记录问题到 `docs/TODAY_PROGRESS.md`

**离开前**：
- [ ] 代码已 push 到云端
- [ ] 任务看板已更新
- [ ] 今日进展已记录（`docs/TODAY_PROGRESS.md`）
- [ ] 配置已文档化
- [ ] 问题已记录

### 状态说明

- `⏳` 待开始 - 可以领取
- `🟡` 进行中 - 已领取，执行中
- `✅` 已完成 - 任务完成，已复盘
