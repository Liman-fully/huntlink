## 📝 如何领取任务

### 步骤

1. **查看任务** - 找到"⏳ 待开始"任务
2. **更新执行者** - 改为你的称呼
3. **更新状态** - 改为"🟡 进行中"
4. **添加领取信息**：
   ```markdown
   **领取人**: [你的称呼]
   **领取时间**: 2026-03-28 18:00
   **预计完成**: 2026-03-28 18:30
   ```
5. **提交更新**：
   ```bash
   git add .task-board.md
   git commit -m "task: 领取任务 [任务 ID]"
   git push origin master
   ```
6. **通知协调者**：
   ```
   @协调者 我已领取任务 [任务 ID]，预计 [时间] 完成
   ```

### 状态说明

- `⏳` 待开始 - 可以领取
- `🟡` 进行中 - 已领取，执行中
- `✅` 已完成 - 任务完成，已复盘
